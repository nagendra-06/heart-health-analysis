# Heart Health Analysis System

## Overview
This project is a machine learning-based system designed to analyze key health indicators and predict the risk of heart disease.  
It demonstrates how data analysis and predictive modeling can assist in early detection and decision-making.

---

## Objectives
- Analyze patient health data
- Identify patterns related to heart disease risk
- Build a predictive model for classification (High/Low risk)

---

## Features
- Generates a dataset with 1000+ records
- Performs data analysis using pandas
- Visualizes patterns using matplotlib
- Implements Logistic Regression for prediction
- Provides risk prediction based on input parameters

---

## Input Parameters
The model uses the following features:
- Age  
- Cholesterol Level  
- Blood Pressure  
- Heart Rate  

---

## Output
- Risk Classification:
  - **High Risk**
  - **Low Risk**

---

## Machine Learning Model
- Algorithm: Logistic Regression  
- Type: Classification  
- Evaluates risk based on multiple health indicators  

---

## Tech Stack
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

---
