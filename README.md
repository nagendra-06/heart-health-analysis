# Heart Health Analysis System

## Description
A machine learning-based system to analyze heart health data and predict risk levels.

## Features
- Generates dataset (1000 records)
- Data analysis using pandas
- Visualization using matplotlib
- Risk prediction using Logistic Regression

## Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Run
pip install pandas matplotlib scikit-learn  
python main.py
